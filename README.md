# Java Lab Programs

This repository contains Java programs from  my MCA lab cycle.  
Each file is named **PROGRAMXX.java**, where `XX` is the program number.

| Program File     | Description                                                                                     |
|------------------|-------------------------------------------------------------------------------------------------|
| PROGRAM01.java   | Program to add, display, sum of command line arguments.                                         |
| PROGRAM02.java   | An arithmetic calculator using switch and command line arguments.                              |
| PROGRAM03.java   | Guess a secret number between 20 and 30 using a do-while loop; display count of attempts.      |
| PROGRAM04.java   | Implement searching and sorting elements in an array.                                          |
| PROGRAM05.java   | Java program to pass an array in different ways.                                               |

| PROGRAM11.java   | Object-oriented program for adding, subtracting, multiplying, and dividing two numbers.        |
| PROGRAM12.java   | Class `Complex` with real and imaginary parts; functions for read, display, add, and multiply. |
| PROGRAM13.java   | Class `CString` with string data member; functions to read, display, compare, add, concatenate. |
| PROGRAM14.java   | Class `Student` with sno, sname, sprogram; functions to read, display, and compare equality.   |
